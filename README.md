# DOCKER-COMPOSER FILES FOR DOCKSERVER

## HOW TO USE [[ EASY AND SIMPLE MODE ]]
```sh
export alias appinstall='curl -s https://app.dockserver.io >> /tmp/app && bash /tmp/app' >> ~/.bashrc
source ~/.bashrc

```
